## Overview

This repository contains the source code for my personal **software engineering portfolio website**.
The platform highlights my work in **backend systems, scalable architectures, and AI-driven applications**, while demonstrating modern frontend engineering practices and performance-optimized UI design.

The goal of this project is to provide a **clean, interactive, and recruiter-friendly experience** that communicates both **technical expertise and engineering mindset**.
